# Интерактивный курс по JavaScript

Базовый курс по JavaScript с теорией, примерами и практическими заданиями с автоматической проверкой.

## Как пользоваться курсом

1. Склонируй репозиторий и установи зависимости:

```bash
git clone <url>
cd js-course
npm install
```

2. Читай теорию в файле `theory.md` каждого урока
3. Открывай файл задания `practice/N/index.js` и пиши решение
4. Проверяй себя, запуская тесты:

```bash
# Все тесты курса
npm test

# Тесты конкретного урока
npm test -- lessons/01-variables

# Тест одного задания
npm test -- lessons/01-variables/practice/1

# Тесты в режиме наблюдения (перезапуск при изменениях)
npm run test:watch
```

## Структура урока

```
lessons/XX-topic/
├── theory.md              # Теория с примерами
└── practice/
    ├── 1/
    │   ├── index.js       # Задание 1 (редактируй)
    │   └── index.test.js  # Тесты к заданию 1 (не редактируй)
    ├── 2/
    │   ├── index.js       # Задание 2
    │   └── index.test.js  # Тесты к заданию 2
    └── ...
```

Каждое задание — отдельная папка. Решай по одному и проверяй тестами.
