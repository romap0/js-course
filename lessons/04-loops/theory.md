# Урок 4. Циклы

Циклы позволяют выполнять блок кода многократно. В JavaScript есть несколько видов циклов.

## Цикл `for`

Самый распространённый цикл. Состоит из трёх частей: **инициализация**, **условие**, **шаг**.

```js
for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4
}
```

```
┌──────────────────────────────────────────┐
│  for (let i = 0; i < 5; i++)             │
│       ▲          ▲       ▲               │
│       │          │       │               │
│  инициализация  условие  шаг             │
└──────────────────────────────────────────┘

Порядок выполнения:
1. Инициализация (один раз)
2. Проверка условия → false? → выход
3. Тело цикла
4. Шаг
5. Переход к пункту 2
```

## Цикл `while`

Выполняет тело, **пока условие истинно**. Удобен, когда количество итераций заранее неизвестно.

```js
let count = 0;
while (count < 3) {
  console.log(count); // 0, 1, 2
  count++;
}
```

## Цикл `do...while`

Гарантирует **хотя бы одно выполнение** тела цикла, потому что условие проверяется после.

```js
let num = 10;
do {
  console.log(num); // 10 (выполнится один раз)
  num++;
} while (num < 5);
```

## `break` — досрочный выход

```js
for (let i = 0; i < 100; i++) {
  if (i === 5) {
    break; // выходим из цикла, когда i === 5
  }
  console.log(i); // 0, 1, 2, 3, 4
}
```

## `continue` — пропуск итерации

```js
for (let i = 0; i < 5; i++) {
  if (i === 2) {
    continue; // пропускаем i === 2
  }
  console.log(i); // 0, 1, 3, 4
}
```

## Перебор массива

```js
const fruits = ["яблоко", "банан", "вишня"];

// Классический for
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// for...of — более современный способ
for (const fruit of fruits) {
  console.log(fruit);
}
```

## Вложенные циклы

```js
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(`${i} × ${j} = ${i * j}`);
  }
}
```

## Типичные ошибки

### Бесконечный цикл

```js
// ПЛОХО — условие никогда не станет false!
while (true) {
  console.log("бесконечность");
}
```

Всегда убеждайся, что условие цикла рано или поздно станет `false`.

### Ошибка на единицу (off-by-one)

```js
const arr = [1, 2, 3];

// ПЛОХО — arr[3] не существует
for (let i = 0; i <= arr.length; i++) { ... }

// ХОРОШО
for (let i = 0; i < arr.length; i++) { ... }
```
