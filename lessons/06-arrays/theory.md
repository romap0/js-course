# Урок 6. Массивы

Массив — упорядоченная коллекция элементов. Один из самых используемых типов данных в JavaScript.

## Создание массива

```js
const fruits = ["яблоко", "банан", "вишня"];
const numbers = [1, 2, 3, 4, 5];
const empty = [];
const mixed = [1, "два", true, null]; // можно смешивать типы
```

## Доступ к элементам

Индексация начинается с **0**:

```js
const arr = ["a", "b", "c"];
arr[0]; // "a"
arr[2]; // "c"
arr.length; // 3
arr[arr.length - 1]; // "c" — последний элемент
```

## Основные методы

### Добавление и удаление

```
┌──────────────────────────────────────────┐
│         unshift ← [a, b, c] → push      │
│                   [a, b, c]              │
│          shift ← [a, b, c] → pop        │
└──────────────────────────────────────────┘
```

```js
const arr = [1, 2, 3];

arr.push(4);      // [1, 2, 3, 4]  — добавить в конец
arr.pop();        // [1, 2, 3]     — удалить с конца
arr.unshift(0);   // [0, 1, 2, 3]  — добавить в начало
arr.shift();      // [1, 2, 3]     — удалить с начала
```

### Поиск

```js
const arr = [10, 20, 30, 20];

arr.indexOf(20);     // 1  — первый индекс
arr.includes(30);    // true
arr.find(x => x > 15);      // 20  — первый подходящий
arr.findIndex(x => x > 15); // 1   — индекс первого подходящего
```

### Преобразование

```js
const nums = [1, 2, 3, 4, 5];

// map — преобразовать каждый элемент
nums.map(x => x * 2); // [2, 4, 6, 8, 10]

// filter — отобрать подходящие
nums.filter(x => x > 3); // [4, 5]

// reduce — свернуть в одно значение
nums.reduce((sum, x) => sum + x, 0); // 15
```

```
map:    [1, 2, 3] → fn(x) → [fn(1), fn(2), fn(3)]
filter: [1, 2, 3] → x > 1? → [2, 3]
reduce: [1, 2, 3] → acc + x → 6
```

### Другие полезные методы

```js
// slice — вырезать копию части массива
[1, 2, 3, 4].slice(1, 3); // [2, 3]

// concat — объединить массивы
[1, 2].concat([3, 4]); // [1, 2, 3, 4]

// join — массив → строка
["a", "b", "c"].join("-"); // "a-b-c"

// sort — сортировка (мутирует массив!)
[3, 1, 2].sort((a, b) => a - b); // [1, 2, 3]

// reverse — разворот (мутирует массив!)
[1, 2, 3].reverse(); // [3, 2, 1]

// flat — «распрямление» вложенных массивов
[1, [2, [3]]].flat(Infinity); // [1, 2, 3]

// every / some — проверка условия
[2, 4, 6].every(x => x % 2 === 0); // true — все чётные
[1, 3, 4].some(x => x % 2 === 0);  // true — хотя бы одно чётное
```

## Деструктуризация

```js
const [first, second, ...rest] = [10, 20, 30, 40];
// first = 10, second = 20, rest = [30, 40]
```

## Spread-оператор

```js
const a = [1, 2];
const b = [3, 4];
const merged = [...a, ...b]; // [1, 2, 3, 4]

// Копия массива
const copy = [...a]; // [1, 2]
```

## Мутирующие vs немутирующие методы

| Мутирует оригинал | Возвращает новый массив |
|---|---|
| `push`, `pop`, `shift`, `unshift` | `map`, `filter`, `slice`, `concat` |
| `sort`, `reverse`, `splice` | `flat`, `flatMap`, `toSorted`, `toReversed` |
